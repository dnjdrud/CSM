# Assets

Place the following image files in this directory before building:

- `icon.png` — App icon (1024x1024 px)
- `splash.png` — Splash screen image (1242x2436 px recommended)
- `adaptive-icon.png` — Android adaptive icon foreground (1024x1024 px)

These are referenced in `app.json`. Expo will resize them automatically for each platform target.
